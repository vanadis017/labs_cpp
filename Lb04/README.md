# Лабораторная работа №4  
**Тема:** Работа с матрицами неправильной формы и динамическим выделением памяти в C++

**Дата выполнения:** 02.03.2026 – 08.03.2026  

---

## Цель работы

Целью лабораторной работы является получение практических навыков работы с непрямоугольными (зубчатыми) матрицами в языке C++, а также освоение:

- динамического выделения и освобождения памяти;
- хранения строк переменной длины;
- вычисления характеристик строк;
- сортировки строк матрицы по заданному критерию.

Особое внимание уделяется принципу работы с указателями: строки матрицы представляются как отдельные динамические массивы, а их сортировка выполняется путем перестановки указателей, а не элементов.

---

## Задание

Необходимо реализовать программу, которая:

1. Создаёт матрицу целых чисел неправильной формы (зубчатую матрицу).
2. Хранит длину каждой строки в отдельном массиве.
3. Вычисляет заданный критерий для каждой строки.
4. Сортирует строки матрицы по этому критерию.
5. Освобождает всю выделенную динамическую память.

---

## Критерий сортировки (вариант 9)

Для данной работы используется следующий критерий:

- **Количество нечётных элементов в строке**

Направление сортировки:

- **по возрастанию**

---

## Требования к реализации

- Использовать только базовые средства языка C++.
- Запрещено использовать:
  - `std::vector`
  - `std::sort`
  - любые контейнеры STL
- Обязательно использование динамической памяти (`new[]`, `delete[]`).
- Сортировка должна выполняться **перестановкой указателей на строки**, а не копированием элементов.
- Все имена переменных и функций должны быть на английском языке.

---

## Основные этапы работы программы

1. Ввод количества строк матрицы.
2. Для каждой строки:
   - ввод её длины;
   - выделение памяти;
   - ввод элементов.
3. Вычисление критерия для каждой строки.
4. Сортировка строк по критерию.
5. Вывод исходной и отсортированной матрицы.
6. Освобождение динамической памяти.
