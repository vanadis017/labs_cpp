# Лабораторная работа №6  
## Использование лямбда?выражений для реализации стратегий обработки данных в C++

---

##  1. Цель работы

Цель лабораторной работы — сформировать практическое понимание **лямбда?выражений в C++** и научиться применять их для построения гибких стратегий обработки данных.

Основная идея — **отделить общий алгоритм обработки массива от конкретных правил фильтрации и преобразования**, передавая эти правила в виде лямбда?функций.

---

##  2. Описание задачи

Программа должна выполнять следующие шаги:

1. Получить входной массив целых чисел.  
2. Отобрать часть элементов по правилу, заданному лямбда?выражением (**filter**).  
3. Преобразовать каждый отобранный элемент с помощью другого лямбда?выражения (**transform**).  
4. Сформировать новый массив внутри функции `process` и вернуть его через `return`.  

Если ни один элемент не подходит под фильтр, функция должна вернуть `nullptr`.
