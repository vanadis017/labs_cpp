# 📘 Лабораторная работа №3  
**Тема:** Работа с матрицами и элементарными функциями   

---

## 🎯 Цели работы

В ходе выполнения лабораторной работы были освоены:

- работа с двумерными массивами (матрицами);
- динамическое выделение памяти (`new[] / delete[]`);
- вычисление элементов матрицы по заданной формуле;
- использование стандартной библиотеки `<cmath>`;
- реализация элементарных функций через ряд Тейлора;
- структурирование программы на модули (`.h` и `.cpp`);
- написание простых автотестов без фреймворков.

---

## 📌 Условие задачи

Реализовать программу, вычисляющую матрицу по формуле:

a_ij =  
(2cos(2i) - sin(2j)) / (i + 1)^2,  если i ≠ j  
i + j,                              если i = j  

---

## ⚙️ Технические требования

- Язык программирования: **C++**
- Тип данных: `double`
- Использование:
  - динамической памяти обязательно

### Ограничения:
- запрещены глобальные переменные
- в реализации через ряд Тейлора запрещено использовать:
  - `sin`, `cos`, `exp` и другие функции из `<cmath>`

## 🔧 Описание программы

Программа выполняет следующие действия:

1. Считывает размеры матрицы `n × m`
2. Выделяет память под матрицу
3. Строит матрицу двумя способами:
   - с использованием стандартных функций (`<cmath>`)
   - с использованием ряда Тейлора
4. Выводит обе матрицы
5. Освобождает память
6. Выполняет тестирование
